Nilbog

Nilbog started out as a "what-if" experiment. It uses mutation observers to revert DOM manipulation on a page. Its only real application is ad-blocking.

  • GitHub
  • Medium article

Alchemize

Alchemize is another mutation observer experiment in the same vein as Nilbog, although much simpler. Developers can use Alchemize to intercept newly added nodes and perform actions on them or trigger simple reactions.

  • GitHub
